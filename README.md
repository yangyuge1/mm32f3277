# mm32f3277
This is a hub for the 17th National College Students Intelligent Car Competition
2022年全国大学生智能汽车大赛平衡信标组别代码+电路原理图
作品视频资料参见个人B站主页：https://space.bilibili.com/666680560

主板（带降压）  
12V锂电池  
摄像头（MT9V03）
